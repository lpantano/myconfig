# most used
* `C-Space` start region + `C-w` delete region + `C-y` paste
* `S-right/left` to move FLAG
* `C-c C-c` add tag
*  `C-c C-a` `l` show list by tags
*  file add to agenda `C-c [`
*  move window `C-x o`
*  remove current window `C-x 0`
*  insert deadline `C-c C-d` or schedule `C-c C-s`
